# reverse 프로젝트

## 프로젝트의 목적
역순 프로그램의 목표는 지정된 입력 파일에서 데이터를 읽어 역순으로 만드는 것입니다. 

## 주요 기능

- 입력
- 출력
- 역순
- 오류 출력

## 사용 기술

파일 읽기 및 쓰기 함수
- fopen(): 파일을 엽니다.
- fclose(): 파일을 닫습니다.
- fgets(): 파일에서 한 줄씩 읽어옵니다.
- fprintf(): 파일에 서식화된 데이터를 씁니다.

연결 리스트 관련 함수
- malloc(): 메모리를 동적으로 할당합니다.
- free(): 동적으로 할당된 메모리를 해제합니다.

문자열 처리 함수
- strcmp(): 두 문자열을 비교합니다.
- strcpy(): 문자열을 복사합니다.
- strlen(): 문자열의 길이를 반환합니다.
