Объявление класса
Класс в Python объявляется с помощью ключевого слова class, за которым следует имя класса.
Конструктор — это специальный метод __init__, который вызывается при создании нового объекта класса. Он инициализирует объект, задавая начальные значения его атрибутов.
Одиночное наследование позволяет одному классу (подклассу) наследовать свойства и методы другого класса (родителя).
Управление доступом к атрибутам класса
•	Публичные атрибуты: доступны извне класса.
•	Приватные атрибуты: обозначаются двумя подчеркиваниями и недоступны напрямую извне класса.
•	Защищенные атрибуты: обозначаются одним подчеркиванием и предназначены для внутреннего использования.
Полиморфизм: использование одинаковых методов для объектов разных классов, упрощая взаимодействие с ними через единый интерфейс.
Утиная типизация: тип объекта определяется по его поведению, а не по наследованию. "Если что-то ведет себя как утка, то это, вероятно, утка".
Для проверки принадлежности объекта к классу –  функция isinstance –  является ли объект экземпляром определенного класса или его подкласса.
