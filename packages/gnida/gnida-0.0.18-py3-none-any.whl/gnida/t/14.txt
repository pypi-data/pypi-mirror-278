Связанные списки – структура данных, в которой каждый элемент содержит ссылку на следующий элемент. Связанные списки могут быть:
•	однонаправленными – элемент имеет ссылку только на следующий
•	двунаправленными – элемент имеет ссылки на предыдущий и следующий элементы

Сравнение с динамическим массивом:
•	Вставка в начало списка (списки быстрее) - O(1) vs O(n)
•	Вставка в конец списка (списки часто быстрее) - O(1) vs O(1) или O(n) в случае расширения массива
•	Удаление из начала списка (списки быстрее) - O(1) vs O(n)
•	Удаление из конца списка (двунаправленные списки быстрее) - O(1) vs O(1) или O(n) в случае динамического массива
•	Доступ к элементу по индексу (динамический массив быстрее) - O(1) vs O(n)
•	Общая сложность вставки/удаления в произвольной позиции - O(n) для списков, O(n) для массива при копировании элементов при необходимости изменения размера.
