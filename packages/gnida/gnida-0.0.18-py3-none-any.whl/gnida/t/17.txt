Алгоритм сортировки вставками – поочередно включает каждый элемент списка в уже отсортированную часть списка. На каждом шаге берется один из элементов неотсортированной части списка и вставляется на правильную позицию в отсортированной части списка, пока весь список не будет отсортирован.

Сложность сортировки вставками:
•	В лучшем случае - O(n)
•	В среднем и в худшем случаях - O(n^2)

Алгоритм быстрого поиска в отсортированном массиве - деление массива пополам и сравнение искомого элемента с элементом в середине. Если элемент найден, поиск завершается. Если искомый элемент меньше среднего элемента, поиск продолжается в левой половине массива, иначе - в правой, пока элемент не будет найден или не останется элементов для проверки.

Сложность быстрого поиска:
•	В отсортированном массиве - O(log n)
•	В неотсортированном массиве - O(n)
