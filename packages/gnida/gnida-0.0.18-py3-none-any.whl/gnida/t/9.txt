map: 
Применяет функцию к каждому элементу последовательности и возвращает новую последовательность с результатами.

filter: 
Фильтрует элементы последовательности по заданному условию и возвращает только те, для которых условие истинно.

reduce: 
Последовательно применяет функцию к элементам последовательности, сводя их к одному значению.

Реализация в Python и пример использования
•	map: 
Используется функция map(), принимающая функцию и последовательность.
•	filter: 
Используется функция filter(), принимающая функцию-условие и последовательность.
•	reduce: 
Используется функция reduce() из модуля functools.

Примеры использования:
map(): Преобразование списка чисел в список их квадратов.
filter(): Отбор только четных чисел из списка.
reduce(): Вычисление суммы всех элементов списка.
