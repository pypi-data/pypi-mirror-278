Вложенные функции:
•	Определяются внутри другой функции.
•	Могут использовать переменные из внешней функции.
Специфика:
•	Доступна в пределах своей внешней функции.
•	Организуют код и ограничивают область видимости вспомогательных функций.
Замыкания:
•	Возникают, когда вложенная функция сохраняет ссылку на переменные внешней функции после её завершения.
•	Запоминают состояние внешней функции для последующего использования.
Специфика реализации в Python:
•	Внутренняя функция использует переменные внешней функции даже после её завершения.
•	Применяются для создания функций с сохраненным состоянием.
